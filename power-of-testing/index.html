¿Qué es un test?

Verificación para saber que el código hace lo que queremos que haga

¿Por qué escribimos tests?

  • ✅ Asegurarnos que el código funciona
  • 🔒 Refactorizar nuestro código sin miedo
  • 📜 Documentación que evoluciona con el código
  • 🔁 Obtener feedback rápidamente
  • 🔍 Aclarar los requerimientos
  • 📐 Para recibir feedback sobre el diseño
